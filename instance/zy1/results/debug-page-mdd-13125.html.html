    歙县

    古徽州 牌坊群 油菜花 新安江 日出 徽墨

    简介 汤显祖的千古绝唱“一生痴绝处，无梦到徽州”道出了的人间仙境原来在徽州，而歙县就是古徽州的发源地。这里是山水幽深处的人间仙境，漫山遍野盛开的油菜花、淡淡炊烟笼罩的马头墙、静静流淌的新安江，日出而作日入而息的原始生活……处处是安静祥和、岁月静好的模样。这里也曾经皇族没落逃难的穷山恶水，却演变出影响大半个中国的文明，从这里走出的商人成为了徽商，从这里走出的戏班成为了京剧的创始者。古徽州是一本厚重的书，需要人去一点点品味，其中的苦辣酸甜，只有阅过的人才知道。走一遍结庐在人境的桃花源，多年后仍会梦见那片水墨丹青的粉墙黛瓦。
