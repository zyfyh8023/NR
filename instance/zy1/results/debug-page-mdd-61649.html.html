    扎金索斯

    简介 扎金索斯岛位于希腊的西部，靠近意大利，属于爱奥尼亚海。这里的中国游客不多，岛屿也相对幽静，岛上地形多样，拥有旖旎多姿的壮美海滩：东南部的沙滩隐蔽在深曲海湾中，海水深蓝静谧；而西岸则是悬崖陡立，另一番有趣的水下世界。此外，岛上生长着超过7,000种花，因此岛屿又被称为“东方之花”(Fioro di Levante)。
