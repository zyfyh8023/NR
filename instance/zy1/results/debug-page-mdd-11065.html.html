    高雄

    台湾 爱河 西子湾 美丽岛

    简介 高雄坐落在台湾西南隅，终年空气潮湿，光照充足，专有的热带季风吹过，绿树成荫，花香袅袅，街道狭窄，宁静的爱河从城中穿过，河畔夜景尤其动人。长长的城垛式防波堤，是莲海路上西子湾经典的标记，这里是最浪漫的夕阳观赏点；爱河入夜后东西两岸各具不同情调的“爱河曼波”与“黄金爱河”咖啡艺文广场，令人不得不放慢脚步；到以客家文化著称的美浓小镇品味乡村的原野气息；在世界最美地铁站之一的捷运“美丽岛”站求婚，秒杀无数菲林。
