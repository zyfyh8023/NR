    砂拉越

    犀鸟之乡 热带雨林 长屋 美食 洞穴 野生动物

    简介 纷繁的野生动物、奇特的长屋、深厚的华人文化、神秘的洞穴、茂密多彩的热带雨林、喷香诱人的娘惹美食……坐落于婆罗洲北部同时也被称为“犀鸟之乡”的砂拉越以它多元的大自然资源吸引着世人。虽没有沙巴海底世界的炫彩，没有吉隆坡的灯红酒绿，但砂拉越以它独特的风情与文化展现在人们面前，它的神秘与优美需要你慢慢挖掘，比如徒步在国家公园追寻大王花和长鼻猴的足迹，比如静静地在海边等待日落，比如去洞穴悄悄拜访穴主蝙蝠……更多的精彩等待你的发现，砂拉越给你最独特的旅行记忆！
