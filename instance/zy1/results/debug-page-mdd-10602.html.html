    同仁

    藏文化发源地 热贡 唐卡 堆绣

    简介 同仁地处青海黄南州，海拔约2500米，藏语称为“热贡”，是唐卡、堆绣等的发祥地，在同仁，几乎每户人家都会画唐卡，每个村落都有精美的藏寺，到同仁的旅行者大多也都是为了热贡艺术而来，同仁县附近的吾屯村、年都乎村、郭麻日村、尕沙日村是热贡艺术的代表地，这里的村民淳朴好客，会热情地邀请游客到自己家里参观唐卡绘制的过程，他们几乎世代从事唐卡艺术，人人都是艺术家，大部分藏区寺院的壁画、堆绣、唐卡、雕塑等都出自热贡艺人之手。
