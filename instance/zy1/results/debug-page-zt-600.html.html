    国际酒店预订及点评用户行为分析报告

    酒店预订 酒店点评 网络预订 42 用户满意度 男女差异

    简介 作为中国最大的旅行分享网站，蚂蜂窝旅行网汇聚了中国最活跃的休闲度假人群，注册用户人数已突破5000万，攻略覆盖全球所有热门目的地，其中酒店、餐饮、景点等点评均来自用户的真实分享，已帮助上亿旅行者制定旅行方案。自2006年起累积了庞大的旅游数据资料库，从数据中更进一步发现网络信息以及产品不全面是阻碍用户进行网上预订的主因。因此，中立客观的海量评论能够吸引消费者通过在线供应商，进行酒店预订。 蚂蜂窝旅行网发布的《2014上半年国际酒店预订与点评用户行为分析报告》结合了2013～2014年数千万的平台数据、全国各地100万条调查样本、覆盖北京、上海、广州、深圳、天津、重庆、成都等150多个城市和地区，受访人群涵盖各个年龄段。通过对用户的关注点进行研究，涵括了对各个城市的不同旅游群体的深入了解，全面分析了当前酒店行业用户点评发展趋势。

    蚂蜂窝攻略中旅行贴士和照片等信息都来自于旅行者的真实分享，感谢蜂蜂对本攻略的贡献
