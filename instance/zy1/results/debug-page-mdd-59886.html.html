    茵莱湖

    茵莱湖

    水上佛塔 独脚渔夫 水上菜园 烤湖鱼 五日市场

    简介 茵莱湖位于缅甸的掸邦高原良水盆地，是缅甸第二大淡水湖。迷人的水上风景和淳朴的人文情怀使得这里成为缅甸的度假胜地，并逐渐成为越来越多外国游客观光的必游目的地。这里有瓦蓝透澈的湖水、单脚划船的渔夫，有梵音袅袅的水上佛寺、潮涨潮落的水上菜园，有历史厚重的岸边塔林、喧嚣忙碌的五日市场，还有田园风情的葡萄酒庄和建于水上的茅草木屋。可以毫不夸张的说，茵莱湖就是缅甸的“马尔代夫”，总是给游客带来啊超乎寻常的旅行体验。如果来缅甸旅行，一定要把茵莱湖作为行程的重要一站，你绝对会不虚此行。
