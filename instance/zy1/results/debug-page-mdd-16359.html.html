    大叻

    疯狂房子 欧洲情调 滴露咖啡 春香湖 避暑胜地 怀旧小火车

    简介 大叻是一座非常复古的小城，充满欧式情调。这里风光明媚，四季如春，百花盛开时，如诗如画。在松林的掩映中，偶尔会看到教堂的尖顶，也可以听到有禅意的古庙铃音。这里旧时是法国人酷爱的避暑胜地，在一栋栋的度假屋里，隔着街道，一辆辆的摩托风驰而过，才让人猛然从异国情调中惊醒过来。
