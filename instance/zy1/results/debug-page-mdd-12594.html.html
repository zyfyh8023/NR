    南投

    日月潭 清境农场 合欢山日出 马拉桑 集集车站 妖怪村 原生态

    简介 四季气候宜人，春夏秋冬，美景如画，晨昏晴雨，景色变幻无穷，这不仅是对日月潭的描述，更是对南投的真实写照。位于台湾地理正中心的南投，是全台湾唯一没有海岸线的县市。这里孕育着得天独厚的自然资源，除了美丽的日月潭和有“小瑞士”之称的清境农场，台湾最高峰玉山、最长河流浊水溪的源头都位于南投县，可谓是天生的观光胜地。怀旧的集集铁路线，雾社抗日纪念碑，古怪又可爱的妖怪村等，又赋予了南投另一番人文情怀。这里独特的水质和气候，造就了喷香的红薯和浓郁的绍兴酒，还有朴实无华的人情民风。虽不曾临海，但是南投却拥有独一无二的摄人美景，犹如世外桃源般，洗去心中浮华，留住一捧碧水清潭。
