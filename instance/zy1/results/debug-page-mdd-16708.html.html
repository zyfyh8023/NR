    亚历山大

    亚历山大

    地中海 埃及夏都 庞贝柱 海滨大道

    简介 从开罗驱车约3小时，湛蓝如洗的地中海就会出现在眼前。而在地中海温柔张开的臂弯里，依偎着它优雅的新娘——亚历山大。与其他埃及城市相比，这里少了些历史的沉重，却多了些轻松惬意，因而是埃及人所热衷的避暑胜地。 当然，除了迷人的海滩景色外，亚历山大自然也有值得驻足的景观：这既有世界七大奇迹之一灯塔的遗迹，也有埃及最大的海港，众多希腊罗马时期的古迹遍布市区各个角落，与阿拉伯风情相映成趣。亚历山大，一道十足的阿拉伯味道的地中海风景线，等待你的到来。
