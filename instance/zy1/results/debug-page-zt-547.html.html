    长白山

    简介 长白山是我国十大名山之一，是与五岳齐名、名光秀丽、景色迷人的关东第一山，为国家首批5A 级风景区，因其主峰白云峰多白色浮石与积雪而得名，素有“千年积雪为年松，直上人间第一峰”的美誉。美丽的长白山三面景色经典但不统一，北坡是沿途独具特色的垂直景观带，让人感受“一山四季”的奇妙，站在环抱天池的火山熔岩畔，仰望一池镜水，天宇辽阔；南坡会路过长白朝鲜族自治县，风景很好，有独特的人文特色，鸭绿江大峡谷、火山遗迹炭化木奇观、高山湿地等壮观景象。
