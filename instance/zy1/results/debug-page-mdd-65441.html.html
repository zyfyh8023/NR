    法属波利尼西亚

    大溪地 波拉波拉岛 潜水 蜜月胜地 观鲨鱼 蓝天 碧水 白沙

    简介 这里被誉为“上帝的恩赐”。清澈见底的海水、碧蓝如洗的天空、柔软细腻的白沙、唯美浪漫的水上屋以及独有的心形岛也使得这里成为了无数情侣梦寐以求的蜜月天堂。在这些群岛里，你可以饱食一顿海鲜大餐、潜水海底与鱼儿一起嬉戏、也可以什么都不做就在沙滩上发呆消磨时间。美妙的假期从这里开始。
