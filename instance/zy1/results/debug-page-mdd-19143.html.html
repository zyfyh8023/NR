    瓦伦西亚

    艺术科学城 法雅节 海鲜饭 斗牛 中央市场 西红柿节

    简介 当清晨的第一缕阳光透过指缝洒落在脸庞，当橘花的芬芳轻轻唤醒你起床，恍惚间你不知自己身在何处。拉开卷帘窗，看到满城古典哥特式建筑，听到耳畔低喃的西班牙语、感受到微咸的海风气息，你便知道这个位于地中海畔被称为“璀璨明珠”的城市有多么美妙。早起去明快而不失整洁的中央市场感受这座城的市井气息；沿老城区小径穿越一个又一个广场只为偶遇古典美学的主教堂；或者跟随着古老的图里亚河床寻找神秘奇幻的艺术科学城……亦或干脆什么都不做，卧在细腻的白沙滩上看地中海潮起潮落。你一定会在心中感叹：竟会有如此古典与现代相得益彰的安逸城市，让你找回久违的宁静和舒适。来瓦伦西亚度假吧，感受西班牙这个热情国度的另外一面，留下只属于你的慵懒回忆。
