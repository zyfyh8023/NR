    神农架 原始森林

    简介 在湖北省的西北部有一处山川交错、峰岭连绵的地方，相传上古的神农氏曾经在这里遍尝百草，为人民治病，由于山高路险，神农氏就搭架上山采药，后来百姓为了纪念神农氏，把这里称为“神农架”。神农架林区山中林密谷深，与世隔绝，完好保存着洪荒时代的风光，动植物资源极其丰富。而最吸引人的，莫过于“野人”的发现。
