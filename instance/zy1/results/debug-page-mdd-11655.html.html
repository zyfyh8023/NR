    开普敦

    南非 桌山 好望角 企鹅岛

    简介 开普敦是南非的立法首都，坐落在桌山脚下的大西洋边，欧洲殖民者就是从这里登陆了非洲大陆；开普敦是南非历史最久远的城市，美丽的桌山山脉环绕着它，来自大西洋的寒风无法影响它，所以这里的气候是全世界最舒适宜人的；难得一见的野生动物在广阔的土地上奔跑，神奇的好望角述说着历史，企鹅岛和海豹岛带来了生灵的惊喜，世界各地的水手和冒险家们带来了不同的文化传统，带给了开普敦多姿多彩的城市风貌。
