    布拉格

    布拉格城堡 老城广场 查理大桥 波西米亚 音乐节 木偶 烤猪肘 人骨教堂

    简介 尼采说：“当我想以一个词来表达音乐时，我找到了维也纳；而当我想以一个词来表达神秘时，我只想到了布拉格。”布拉格就是这样一个神秘的城市。站在城市的高地，从高处俯瞰，高高低低的塔尖，毗连成一片塔林。在阳光照耀下，“百塔”显得金碧辉煌，犹如撒上了一层金粉一般。城内宏伟的建筑和广场，甚至是一砖一瓦都透露着她古老的神秘，由这种神秘滋生的浪漫更是让人沉醉于此。这个欧洲最美丽的城市，以她独特的魅力吸引着世界各地的游客来此领略她的风采。
