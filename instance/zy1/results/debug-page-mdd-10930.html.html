    圣迭戈

    中途岛 墨西哥风情 美国 阳光之城

    简介 美墨边境的圣迭哥是彩色的，北美的热情在这里显得十分迷人。圣迭戈，光是名字念起来就有股特别的味道，仿佛会在口里激起迷人的芳香。难怪有许多人将之列入40岁以前的旅行计划。阳光、沙滩、军舰和公园，组成了绝美的度假天堂。而静静地永久性地停泊在海边的庞大的航空母舰旁的水兵与护士之吻雕塑，则为圣迭戈赋予了一丝浪漫。一切景物，一切韵律，在由近及远、由蓝及白的海洋衬托下，显得是那么和谐与亲切，那么的浪漫与宁静，那么的如诗如画。阳光之城圣迭戈，邀你与我一同来这做时间的盗贼，忘却步履匆忙，静享难得的闲适。
