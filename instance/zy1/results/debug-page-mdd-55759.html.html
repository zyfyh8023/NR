    鸡鸣岛

    海岛石崖 海草房 海鸥 出海打渔

    简介 如果有一天，你发现自己在一个人烟稀少的孤立海岛上，只可泛舟出行，没有路灯，只有雨水喝，还睡在海草搭建的古老房屋里，但是你有淳朴的邻居，可以醉卧鲜花丛中，可爱的海鸥为你唱歌，温柔的涛声伴你入眠……如此，你是否还会愿意醒来？ 然而这不是梦，因为这就是现实的模板，在胶东湾沿海地带，位于荣成市港西镇东北角的海域，有一个不为人知的古老海岛——鸡鸣岛，因2013年一场大型亲子秀节目，犹如待阁闺中的少女，正缓缓地揭开面纱。在这里，没有浩瀚壮观的风景，没有令人惊艳的奇观，也没有刺激诡秘的探险，有的只是简单古朴的生活，安宁淡然的气氛，以及那平凡而又暖心的笑脸。在这里，你感觉不到陆地的躁动，更远离世间的喧嚣，心底只有海那抹蓝色荡漾。在这里，时间显然已经没有存在的必要，你只需要一个人带着一颗心，闭眼感受这忘却时光的欢愉或寂寞。
