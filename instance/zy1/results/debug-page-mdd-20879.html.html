    齐云山

    横江漂流 飞云亭 梦真桥 月华街 太素宫

    简介 齐云山位于安徽黄山市休宁县，是一处以摩崖石刻、道教文化和丹霞地貌为特色的山岳景区，奇峰怪岩飞涧与河湖泉潭瀑构成了一幅峭拔明丽的自然图画。全山地质构成层积砂岩，赤如朱砂，灿若红霞，为典型的“丹霞地貌”；这里是全国四大道教圣地之一，道教绘画、摩崖石刻和碑刻星罗棋布，几乎峰峰有题词、洞洞有刻铭；“危崖神奇峰，绝壁布幽洞，岩额泻银瀑，幻景变无穷”整合出一幅大写意山水画。
