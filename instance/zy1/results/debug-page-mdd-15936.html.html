    哈瓦那

    朗姆酒 切格瓦拉 海明威 顶级雪茄 西班牙殖民 国会大厦 老爷车

    简介 哈瓦那是古巴绝对的旅游中心，这座海港城市由于几位名人而独具风韵——海明威、切格瓦拉、卡斯特罗，在这里他们留下的故事和足迹，至今仍然被人们讲述，并将一直被人们保存下去；在这里，古典与现代、新大陆与旧大陆、白色与黑色、下里巴人与阳春白雪，一切充满碰撞的元素，都有迹可循。所有看似独立的个体，一起都在这个阳光明媚、生机勃勃的热带港口和谐的统一起来。
