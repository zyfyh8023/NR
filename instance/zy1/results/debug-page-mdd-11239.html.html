    贵阳

    甲秀楼 酸汤鱼 丝娃娃 青岩古镇

    简介 贵阳四周青山环绕，是名副其实的山城，也称“林城”。作为“高原第二春城”，相比于“第一春城”昆明的热情浓艳，她则更加灵秀动人。尤以喀斯特地貌著称的贵阳，好山好水无处不在，周边葱翠山峰和碧湖溶洞连绵不尽。而除却山清水秀，惬意自在是贵阳的另一张名片。舒缓的生活节奏让整个城市从容不迫，避免了大都市的紧张感。但她绝对不乏活泼的一面。等待入夜，从小吃街到酒吧街都热闹非凡，显示出别样的生机与活力。更重要的是，贵阳还扮演着中转城市的角色。只要来贵州游玩，你一定会在这里有所停留。
